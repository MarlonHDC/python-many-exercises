### 🔧 Instalação

É uma IDE fácil de instalar. Para isso basta você baixar o arquivo "Comunidade" e clicar em avançar... avançar... até finalizar a instalação.

Baixe o arquivo Visual Studio versão "Comunidade" e depois:

```
Avançar... Avançar...
```

E repita:

```
Até finalizar
```
